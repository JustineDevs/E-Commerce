# Architecture Decision Records (ADRs)

**Accepted ADRs** (frozen decisions) live in **`docs/archived/adr/`**.

Use this folder for **new** proposals:

1. Add a numbered file (e.g. `0004-your-topic.md`) with `Status: Proposed`.
2. After review, set `Status: Accepted` and move the file to `docs/archived/adr/`.

See `docs/archived/README.md` for the archive policy.
